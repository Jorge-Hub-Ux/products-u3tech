# Guía rápida (ES)

Pasos para ejecución del codigo

## Requisitos
- Node.js 18+ y npm (el repo define `npm@11.6.2` como gestor)
- Angular CLI disponible vía `npx` (no es necesario instalarlo globalmente)

## Instalación
```bash
npm install
```

## Ejecutar en desarrollo
```bash
npm start
# o
npx ng serve
```
Abre `http://localhost:4200/` para ver la app; el servidor recarga al guardar cambios.

## Ejecutar pruebas unitarias
```bash
npm test
```
Lanza `ng test` con Vitest y recarga las pruebas en modo watch.
